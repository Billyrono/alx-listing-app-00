# ALX Listing App

## Description

This project is the foundation for an Airbnb clone listing page built with Next.js, TypeScript, TailwindCSS, and ESLint.

## Project Structure

- `components/` → Reusable components (Card, Button)
- `interfaces/` → TypeScript interfaces
- `constants/` → Configurations and constants
- `public/assets/` → Images and SVGs

## Run Locally

```bash
npm install
npm run dev
```
